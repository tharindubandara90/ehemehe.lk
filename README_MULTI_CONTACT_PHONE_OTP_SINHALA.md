# Multiple Contact Phone OTP Verification

- Ad එකකට අඩුම ගණනේ phone number 1ක් අනිවාර්යයි.
- කැමති නම් උපරිම phone numbers 5ක් add කළ හැක.
- සෑම number එකක් ඉදිරියේම Verify button එකක් ඇත.
- එක් එක් number එකට වෙනම SMS OTP යවයි.
- සියලු numbers verify කරන තුරු Publish Ad ක්‍රියා නොකරයි.
- Duplicate numbers allow නොකරයි.
- Verified numbers සියල්ල ad detail page එකේ click-to-call links ලෙස පෙන්වයි.
- පළමු verified number එක primary Call Now number එක වේ.
