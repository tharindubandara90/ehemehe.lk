# Internal Public Ads management

මෙම update එකෙන්:
- Admin sidebar එකේ Public Ads button එක external /browse link එකක් නෙමෙයි.
- Public Ads දැන් admin dashboard ඇතුළේ internal tab එකක්.
- Admin / Moderatorලාට ads view/edit/approve/reject/delete කරන්න පුළුවන්.
- Permission නැති moderatorලාට buttons disabled/hide වෙනවා.
- /browse public URL එක ads list නොපෙන්වයි. ඒක admin වෙත redirect වෙනවා.

Run:
```bash
npx serve public
```

Admin:
```text
http://localhost:3000/admin
```
