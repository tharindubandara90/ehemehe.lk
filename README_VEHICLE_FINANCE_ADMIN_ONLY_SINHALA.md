# Vehicle Finance admin-only settings update

Updated:
- Vehicle Finance Settings moved to a separate admin sidebar option.
- User/Post Ad page does not show or ask for finance rate.
- User only enters vehicle price; down payment and monthly payment are calculated automatically.
- Admin controls down payment %, annual finance rate %, term months, and finance company phone.
- Admin can still manually edit vehicle ads from Ad Management.

Run:
```bash
npm install
npm run dev
```

Admin:
```text
http://localhost:3000/admin
```

After replacing files, press Ctrl + F5 in the browser.
